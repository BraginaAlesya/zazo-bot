## Шаг 1 — Добавь usernames команды

Открой `bot.py`, найди раздел REGISTRARS и добавь всех:

```python
REGISTRARS = {
    "alesyabragina": "Алеся",
    "username_zheni": "Женя",    # ← реальный username Жени без @
    "username_olesi": "Олеся",   # ← реальный username Олеси без @
}
```

---

## Шаг 2 — Запуск на Railway (5 минут)

1. Зайди на railway.app → войди через GitHub
2. New Project → Empty Project
3. + Add Service → загрузи папку с файлами
4. Variables → добавь: BOT_TOKEN = токен ZBot от @BotFather
5. Settings → Start Command: python bot.py
6. Deploy — готово!

---

## Команды бота

/билет   — выдать новый билет (4 шага: ФИО → места → сумма → контакт)
/список  — все выданные билеты с итогами
/отмена  — отменить текущий диалог
